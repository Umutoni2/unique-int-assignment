# UniqueInt Assignment – README

## Overview

This Node.js program reads integers from text files, removes duplicates, sorts them in ascending order, and saves the results. It uses a **linked list** to keep numbers unique and sorted.

Outputs go to **`hw01/sample_output/`** and are automatically renamed if a file already exists.

---

## How to Run

### Single File

```powershell
node UniqueInt.js <inputFile> <outputFile>
```

Example:

```powershell
node UniqueInt.js ../../sample_inputs/sample_input_03.txt out1.txt
```

### Batch Mode

```powershell
node UniqueInt.js <inputFolder>
```

Example:

```powershell
node UniqueInt.js ../../sample_inputs
```

All outputs are saved in `hw01/sample_output/`.
